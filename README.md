# 🚗 Rent-a-Car Full Stack System

Sistema web fullstack para la gestión de alquiler de vehículos, desarrollado con ASP.NET Core en el backend y tecnologías web en el frontend.

## 📌 Descripción

Este proyecto permite administrar un sistema de renta de autos donde se pueden gestionar vehículos, clientes y procesos de alquiler. Está diseñado como una aplicación CRUD completa con arquitectura cliente-servidor.

El objetivo principal del proyecto es demostrar habilidades en desarrollo backend con C#, consumo de API y manejo de bases de datos.

---

## 🛠️ Tecnologías utilizadas

### Backend
- C#
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server

### Frontend
- HTML5
- CSS3
- JavaScript
- Bootstrap

### Herramientas
- Git & GitHub
- Swagger (documentación de API)
- Visual Studio

---

## ⚙️ Funcionalidades

- 🔹 Registro, edición y eliminación de vehículos
- 🔹 Gestión de clientes
- 🔹 Sistema de alquiler de vehículos
- 🔹 API REST para comunicación frontend-backend
- 🔹 Validación de datos
- 🔹 Consulta de información en tiempo real

## 🚀 Cómo ejecutar el proyecto

### Backend
1. Abrir la solución en Visual Studio
2. Configurar la cadena de conexión en `appsettings.json`
3. Iniciar el proyecto 

### Frontend
1. Abrir `index.html` en el navegador  
   o usar Live Server en VS Code
